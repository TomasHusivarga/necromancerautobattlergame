## Battle Screen Prompt Base

```text
Clean PC 16:9 production-readability mockup for a necromancer autobattler RTS battle screen. Old-RTS high-angle orthographic battlefield. Main battlefield fills the top area, bottom spell bar fills the bottom. Use broad terrain patches and large empty combat spaces. Show 3-5 zombie squad groups versus 3-5 human enemy squad groups. Each squad should read as one grouped formation shape with only a few visible figure silhouettes, not many individual units. Show two spell effects maximum. Bottom spell bar has five large spell cards/icons. Use the approved graphic big-shape gothic style. No fake text, no painterly RTS screenshot look, no realistic terrain texture, no dense rubble, no huge crowd.
```
